# Smart Resume Ranker

Ranks resumes based on their similarity to a job description using NLP.

## ✨ Features
- Extracts text from PDF resumes
- Computes similarity using TF-IDF and cosine similarity
- Outputs ranked results to CSV

- Python 3.x
